# Byte Mark ξ( ✿＞◡❛)! 🔍

## 📖 项目简介

Byte Mark 是一个轻量级的**十六进制报文索引工具**，专为快速解析、标记和搜索二进制数据而设计。通过直观的可视化界面，帮助开发者轻松查看和分析十六进制报文中的每个字节及其位置索引。

## ✨ 核心功能

### 🎯 当前版本（Web端）

- **智能解析**：支持多种十六进制输入格式
  - 空格分隔：`00 01 ff 19 23`
  - 逗号分隔：`00,01,ff,19,23`
  - 连续字符串：`0001ff1923`
  - 带前缀：`0x00 0x01 0xff`

- **灵活显示**：三种视图模式
  - 📊 **网格视图**：卡片式展示，适合快速浏览
  - 📋 **表格视图**：详细展示十六进制、十进制、二进制值
  - 📝 **列表视图**：简洁的文本列表格式

- **自定义配置**
  - 自定义起始索引（从任意位置开始编号）
  - 多种显示格式：十六进制 / 十进制 / 两者兼有
  
- **字节搜索**：快速定位特定字节值，高亮显示所有匹配项

- **实时统计**：自动计算总字节数和有效字节数

- **便捷操作**
  - 一键复制结果到剪贴板
  - 清空重置功能
  - 响应式设计，适配不同屏幕尺寸

## 🚀 使用方法

1. **打开应用**：直接在浏览器中打开 `ByteMark.html` 文件
2. **输入报文**：在文本框中粘贴或输入十六进制报文
3. **配置选项**：设置起始索引和显示格式
4. **查看结果**：下方实时显示标号后的字节数据
5. **搜索字节**：使用搜索框查找特定字节值
6. **切换视图**：根据需要切换网格/表格/列表视图
7. **导出结果**：点击"复制结果"按钮导出数据

## 💻 技术栈

- **前端**：纯 HTML + CSS + JavaScript（无依赖）
- **兼容性**：现代浏览器（Chrome、Firefox、Edge、Safari）
- **部署**：无需服务器，可直接本地运行
